# Netflix-clone

● Mobile IOS application, clone of the Netflix app, based on Model-View-ViewModel architecture.
● Learned about customizable view layouts, delegate design patterns and precise UI display.
● Uses CoreData as a database to store the downloaded movies, with real time updates on the download page.
